# AI Sentinel Phase 1: Ghost AI Detection Foundation

## Goal
Establish a baseline Ghost AI detection system with real-time analysis and risk categorization.

## Key Features
- **Network Log Analysis**: Real-time processing of structured logs.
- **Rule-based Pre-screening**: Fast-track for approved domains and immediate detection of known malicious URLs.
- **GPT-4 Risk Assessment**: Intelligent analysis of request payloads for intent and risk.
- **Sensitive Data Scanner**: Regex-based detection for IBANs, PII, and financial data.
- **Dashboard v1**: Core detection metrics and risk categorization visualization.
- **Department Profiling**: Sensitivity mapping for different business units.
