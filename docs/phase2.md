# AI Sentinel: Phase 2 (Agentic Orchestration)

## Goal
Transition to an agentic system capable of autonomous reasoning and automated response (SOAR).

## Key Features
- **LangGraph Agent**: A stateful graph for security analysis and tool use.
- **Multi-Tool Reasoning**: Integrating VirusTotal, domain reputation, and sensitive content scanning into one flow.
- **SOAR Engine**: Automated mitigation triggers (Webhooks).
- **Incident Response Tab**: Dedicated UI for tracking automated actions and alerts.
- **Deterministic Overrides**: Guaranteed "Critical" risk categorization for confirmed high-confidence threats.
- **Automated Notifications**: Mock Slack/Teams broadcast system.
